# AIthar – Financial Waste Detection using GenAI

AIthar is an AI-powered solution designed to help high-spending entities
identify repeated spending and price discrepancies across contracts,
invoices, and payment orders.

## Problem
Financial audit teams struggle to detect invisible financial waste due to:
- Fragmented financial documents
- Manual review processes
- Lack of intelligent comparison between similar services

## Solution
AIthar uses Generative AI on AWS to:
- Analyze contracts, invoices, and payment orders
- Detect repeated spending and price discrepancies
- Generate clear, explainable text-based insight reports

## AWS Services Used
- Amazon S3 – Secure document storage
- AWS Lambda – Serverless processing
- Amazon Bedrock – Generative AI analysis
- (Future) Amazon Textract, Amazon QuickSight

## Use Case
Designed for internal audit teams in high-spending entities such as:
- Infrastructure
- Public services
- Large enterprises
